# 📝 Board (게시판 어플리케이션 실습)
**대규모 트래픽 환경에서의 데이터 정합성과 성능 최적화를 고려한 게시판 서비스입니다.**
단순한 기능을 넘어 수만 건의 동시 요청이 발생하는 상황을 가정하여 Kafka를 통한 이벤트 기반 아키텍처와 Redis 캐싱 전략을 적용해 시스템 안정성을 높이는 데 집중했습니다. <br/>
